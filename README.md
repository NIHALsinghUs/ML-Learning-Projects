# Machine Learning Basics with Scikit-learn

This project is a beginner-friendly collection of Machine Learning examples built using **Python**, **Pandas**, **NumPy**, and **Scikit-learn**. It demonstrates the complete machine learning workflow, from data preprocessing to model training and evaluation.

## Features

* Label Encoding
* One-Hot Encoding
* Standard Scaling
* Min-Max Scaling
* Train-Test Split
* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree Classifier
* Accuracy Score
* Precision Score
* Recall Score
* F1 Score
* Confusion Matrix
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Purpose

The goal of this project is to help beginners understand the fundamentals of Machine Learning through simple examples and a menu-driven Python program. Each topic can be run individually, making it easy to learn preprocessing techniques, train different models, and evaluate their performance.

This project is ideal for students, beginners, and anyone starting their Machine Learning journey.
